初次使用Hexo的踩坑记录

Posted on 2020-04-25 Edited on 2020-04-26

下载node.js

  • 建议下载LTS版本，我下载了最新的Current版本，导致在部署到github上是出现了各种错误，所以谨慎下载。官网下载很慢，可以挂vpn或者找网友下好的包。

  • 需要确认git的安装，没有的话需要下载，之后的命令建议用git bash打开

  • npm慢的话可以用cnpm挂淘宝的镜像

初体验之命令

  • node -v // 查看node.js版本
  • npm -v // 查看npm版本
    …
  • npm install -g cnpm --registry=http://registry.npm.taobao.org// 安装淘宝的cnpm
  • cnpm install -g hexo-cli// 安装hexo框架
  • hexo init blog// 在当前文件夹创建blog文件并生成hexo框架
  • hexo new "title"// 创建新的文章
  • cd \blog\source\_posts//进入该文件夹找到md文件，在这就可以编辑你的文章了
  • hexo clean// 清理一下
  • hexo g// 生成静态页面
  • hexo s // 打开server
  • localhost：4000// 查看你的本地blog
  • 配置blog目录下的_config.yml文件
  • hexo d // 部署到github
  • enjoy your time
